# Backend

This project currently has **no custom backend server**. The frontend
communicates directly with **Firebase** (Authentication, Firestore, and
Storage) as a backend-as-a-service.

This folder is reserved for a future Node.js / Express API, should one be
added (e.g., for server-side logic, third-party integrations, or admin
tooling that shouldn't live in the client).

## Planned setup (once backend code is added)

```bash
cd backend
npm install
npm run dev
```
